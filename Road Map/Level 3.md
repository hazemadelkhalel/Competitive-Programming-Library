# Level 3
Duration: 30 rokba days or more.

Target: be able to solve CF-div2 ABCD(E)

### Content
* SCC + Tarjan
* LCA
* Euler's Graph
* Mobius + Linear Diophantine Equations
* DP (optimizations - ranges - bitmask - digits ... etc.)
* Segment tree
* Segment tree with lazy propagation
* Matrix Exponentiation
* Trie 
* KMP algorithm - Z algorithm
* Hashing Rabin-Karp algorithm
* Suffix Array
* Game Theory
* Small to Large Trick
* Mo Algorithm
* Convex Hull Trick
* Matching Kuhn + Hopcroft
* Xor Basis

## Timeline
To use this plan:
 - For coaches: each point of the content takes a week (1 session for teaching + 1 session for upsolving the sheat)
 - For trainees: try to master each point in at most (3-5) days (learning + finishing the sheet)
